## 部署配置总结

### 文件结构
```
deploy/
├── .env                    # 默认环境配置
├── .env.dev                # 开发环境配置
├── .env.test               # 测试环境配置
├── .env.prod               # 生产环境配置
├── docker-compose-base.yml # 中间件配置（PG17、Redis Stack、MinIO、Ollama）
├── docker-compose.yml      # 应用配置（Python、Java、前端）
├── start.sh                # 启动脚本
├── stop.sh                 # 停止脚本
├── status.sh               # 状态检查脚本
├── chuangxin/build-arm64.sh # 信创ARM构建脚本
├── k8s/finrag4j-deployment.yaml # K8s部署配置
└── offline-package/build-offline-package.sh # 离线打包脚本
```

### 环境变量配置要点
| 配置项 | 值 | 说明 |
|--------|-----|------|
| POSTGRES_PORT | 5432 | PostgreSQL端口 |
| REDIS_PORT | 6379 | Redis端口 |
| MINIO_PORT | 9000 | MinIO端口 |
| OLLAMA_BASE_URL | http://192.168.222.1:11434 | 外部Ollama地址 |
| JAVA_SERVICE_PORT | 8080 | Java服务端口 |
| FRONTEND_PORT | 80 | 前端端口 |

### 使用命令
```bash
# 启动全部服务（自动加载.env）
docker-compose up -d

# 指定环境启动
docker-compose --env-file .env.prod up -d

# 仅启动中间件
docker-compose -f docker-compose-base.yml up -d
```
