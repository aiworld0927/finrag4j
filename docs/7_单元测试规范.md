# 7_单元测试规范

## 1. 测试框架

- Java: JUnit 5 + Mockito
- Python: pytest + unittest

## 2. 测试覆盖率

- 核心代码覆盖率 ≥ 80%
- 公共API覆盖率 ≥ 100%

## 3. 测试编写规范

### 3.1 测试命名
- 方法名：test_类名_方法名_场景

### 3.2 测试结构
- Given：准备测试数据
- When：执行被测方法
- Then：断言结果

## 4. 测试执行

### 4.1 单元测试
mvn test / pytest

### 4.2 代码覆盖率
- Java: JaCoCo
- Python: coverage.py
