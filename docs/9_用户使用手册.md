# 9_用户使用手册

## 1. 系统概述

FinRag4j是一款面向金融行业的大模型RAG应用平台，支持文档解析、智能检索、问答服务等功能。

## 2. 快速入门

### 2.1 登录系统
访问 http://localhost:8080，使用用户名密码登录

### 2.2 上传文档
- 进入文档管理页面
- 点击上传按钮
- 选择本地文档文件
- 等待解析完成

### 2.3 检索文档
- 进入检索页面
- 输入检索关键词
- 点击搜索按钮
- 查看检索结果

### 2.4 智能问答
- 进入问答页面
- 输入问题
- 点击提问按钮
- 查看回答结果

## 3. 高级功能

### 3.1 Agent流程
- 进入Agent页面
- 创建工作流
- 配置节点
- 执行工作流

### 3.2 权限管理
- 进入权限管理页面
- 创建角色
- 分配权限
- 管理用户
