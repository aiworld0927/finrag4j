## PPT制作规范

### 设计规范

| 元素 | 规范 |
|------|------|
| 字体 | 中文：微软雅黑，英文：Arial |
| 字号 | 标题：32号，正文：24号 |
| 配色 | 主色调：蓝色，辅助色：灰色 |
| 风格 | 简洁专业，图文并茂 |

### 内容规范

- 每页内容不宜过多，重点突出
- 使用图表、图示代替大段文字
- 保持页面风格统一
- 重点内容用颜色或动画突出
