# FinRag4j 信创国产化部署手册

## 文档概述

本文档提供FinRag4j在国产信创环境的部署指南，支持麒麟操作系统、鲲鹏处理器等国产化平台。

**文档版本**: v1.0.0  
**适用环境**: 国产信创环境

## 2. 部署前准备

### 2.1 安装Docker

**麒麟系统安装Docker**:

```bash
# 安装依赖
yum install -y yum-utils device-mapper-persistent-data lvm2

# 添加Docker源
yum-config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

# 安装Docker
yum install -y docker-ce docker-ce-cli containerd.io

# 启动Docker
systemctl start docker
systemctl enable docker
```

### 2.2 安装JDK

**安装麒麟JDK**:

```bash
# 安装OpenJDK 11
yum install -y java-11-openjdk java-11-openjdk-devel

# 配置环境变量
export JAVA_HOME=/usr/lib/jvm/java-11-openjdk
export PATH=$JAVA_HOME/bin:$PATH
```

### 2.3 安装PostgreSQL

**安装国产数据库适配版本**:

```bash
# 使用达梦数据库
# 或使用PGVector ARM64版本
docker pull pgvector/pgvector:pg16
```

## 3. ARM64镜像构建

### 3.1 构建Java服务镜像

```bash
cd backend/finrag4j-java

# 使用ARM64 Dockerfile
docker build -f Dockerfile.arm64 -t finrag4j/java-service:1.0.0-arm64 .
```

### 3.2 构建Python服务镜像

```bash
cd backend/finrag4j-python

# 使用ARM64 Dockerfile
docker build -f Dockerfile.arm64 -t finrag4j/python-service:1.0.0-arm64 .
```

### 3.3 构建前端镜像

```bash
cd frontend

# 使用ARM64 Dockerfile
docker build -f Dockerfile.arm64 -t finrag4j/frontend:1.0.0-arm64 .
```

## 4. 国产数据库适配

### 4.1 达梦数据库配置

```yaml
# application-dm.yml
spring:
  datasource:
    driver-class-name: dm.jdbc.driver.DmDriver
    url: jdbc:dm://localhost:5236/finrag4j
    username: SYSDBA
    password: SYSDBA
```

### 4.2 人大金仓配置

```yaml
# application-kingbase.yml
spring:
  datasource:
    driver-class-name: com.kingbase8.Driver
    url: jdbc:kingbase8://localhost:54321/finrag4j
    username: system
    password: system
```

## 5. 部署执行

### 5.1 使用部署脚本

```bash
cd deploy/chuangxin
chmod +x build-arm64.sh
./build-arm64.sh
```

### 5.2 启动服务

```bash
cd deploy
docker-compose -f docker-compose.yml -f docker-compose.arm64.yml up -d
```

### 5.3 验证部署

```bash
# 检查服务状态
docker-compose ps

# 检查日志
docker-compose logs -f java-service
```

## 6. 性能优化

### 6.1 JVM参数优化

```bash
# 针对ARM64处理器优化
JAVA_OPTS="-Xms4g -Xmx4g -XX:+UseG1GC -XX:ParallelGCThreads=8"
```

### 6.2 数据库优化

```sql
-- PostgreSQL ARM64优化
shared_buffers = 4GB
effective_cache_size = 12GB
maintenance_work_mem = 1GB
checkpoint_completion_target = 0.9
wal_buffers = 16MB
```
