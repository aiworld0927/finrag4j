# 6_开发手册

## 1. 开发环境

### 1.1 环境要求
- JDK 21+
- Python 3.10+
- Maven 3.9+
- uv 0.1+

### 1.2 开发工具
- IntelliJ IDEA 2024+
- VS Code

## 2. 代码规范

### 2.1 Java代码规范
- 遵循阿里巴巴Java开发手册
- 类名使用大驼峰
- 方法名使用小驼峰
- 常量使用全大写下划线分隔

### 2.2 Python代码规范
- 遵循PEP8规范
- 使用black格式化代码
- 使用isort排序导入

## 3. 开发流程

### 3.1 Git工作流
- main：生产分支
- develop：开发分支
- feature/*：功能分支
- bugfix/*：修复分支

### 3.2 代码提交规范
- feat: 新功能
- fix: 修复bug
- docs: 文档更新
- refactor: 重构
- test: 测试

## 4. 本地运行

### 4.1 启动依赖服务
docker-compose up -d

### 4.2 启动Python服务
cd backend/finrag4j-python
uv run python main.py

### 4.3 启动Java服务
cd backend/finrag4j
mvn spring-boot:run
