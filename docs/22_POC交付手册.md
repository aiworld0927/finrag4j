## 2. 标准演示流程

### 2.1 演示环境检查

```bash
# 1. 检查服务状态
docker-compose ps

# 2. 检查服务健康
curl http://localhost:8080/actuator/health

# 3. 检查数据库连接
docker exec -it finrag4j-postgres psql -U postgres -c "SELECT 1"
```

### 2.2 演示流程脚本

**场景1: 知识库管理演示**

1. 登录系统（管理员账号）
2. 创建知识库"金融法规库"
3. 上传测试文档（PDF/Word）
4. 查看文档解析状态
5. 预览文档内容

**场景2: RAG问答演示**

1. 进入问答界面
2. 提问："什么是信贷业务？"
3. 查看AI回答
4. 展示来源引用（文件名、页码）
5. 继续追问："有哪些合规要求？"
6. 展示多轮对话能力

**场景3: 合规自查演示（商业版）**

1. 进入Agent中心
2. 选择"监管合规自查"
3. 上传业务文档
4. 执行合规检查
5. 查看风险分级结果
6. 导出整改报告

**场景4: 工作流演示（商业版）**

1. 进入工作流设计器
2. 展示可视化编排
3. 创建审批流程
4. 触发流程执行
5. 处理待办任务

**场景5: 审计日志演示（商业版）**

1. 进入审计日志页面
2. 按时间范围查询
3. 按用户筛选
4. 导出审计报表

## 4. 常见问题应对

### Q1: 支持哪些大模型？

A: 支持Ollama部署的开源模型（Qwen、ChatGLM等），也支持vLLM加速部署。商业版支持对接商业大模型API。

### Q2: 数据安全如何保障？

A: 
1. 私有化部署，数据不出域
2. 多租户物理隔离
3. 全链路审计日志
4. RBAC细粒度权限控制

### Q3: 与其他产品相比有什么优势？

A:
1. 金融垂直：预置金融知识库和合规检查
2. 开源免费：基础功能完全开源
3. 国产信创：支持麒麟、鲲鹏等国产环境
4. 可扩展：工作流编排、Agent定制

### Q4: 部署需要多长时间？

A: 单机部署约2小时，集群部署约1天。提供离线部署包，支持无外网环境部署。
