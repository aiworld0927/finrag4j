# 9_用户使用手册

## 1. 系统概述

FinRag4j是一款面向金融行业的大模型RAG应用平台，支持文档解析、智能检索、问答服务等功能。

## 2. 快速入门

### 2.1 登录系统
访问 http://localhost:5173，使用用户名密码登录

默认账号：admin / admin

### 2.2 工作台
登录后进入工作台页面，展示系统概览信息：
- 知识库数量统计
- 文档数量统计
- 今日问答数
- 模型使用率
- 快速操作按钮
- 最近文档列表
- 问答趋势图表
- 系统通知

### 2.3 上传文档
- 进入「知识库管理」->「文档管理」页面
- 点击「上传文档」按钮
- 选择或拖拽本地文档文件（支持PDF、Word、Excel、TXT）
- 选择目标知识库
- 点击「开始上传」
- 等待解析和向量化完成

### 2.4 检索文档
- 进入「文档管理」页面
- 使用搜索框搜索文档
- 使用状态筛选筛选文档
- 点击文档行的「预览」按钮查看文档内容
- 支持下载和删除操作

### 2.5 智能问答
- 进入「问答中心」->「RAG问答」页面
- 在输入框输入问题
- 选择目标知识库（可选）
- 按 Ctrl+Enter 或点击发送按钮
- 查看回答结果和参考来源

## 3. 知识库管理

### 3.1 创建知识库
- 进入「知识库管理」页面
- 点击「创建知识库」按钮
- 填写知识库名称、编码、描述
- 选择默认模型
- 设置相似度阈值和召回数量
- 点击「保存」

### 3.2 管理知识库
- 查看知识库列表
- 编辑知识库配置
- 删除不需要的知识库
- 查看知识库统计信息

## 4. 对话历史

### 4.1 查看对话
- 进入「问答中心」->「对话历史」页面
- 浏览所有历史对话
- 双击对话卡片打开对话详情
- 使用搜索框搜索对话

### 4.2 管理对话
- 支持批量选择和删除
- 点击「新对话」创建新会话

## 5. Agent中心

### 5.1 信贷材料抽取
- 进入Agent中心页面
- 点击「信贷材料抽取」卡片
- 选择抽取模板
- 上传文档
- 点击「执行抽取」
- 查看抽取结果和置信度

### 5.2 监管合规自查
- 进入Agent中心页面
- 点击「监管合规自查」卡片
- 输入报告名称
- 选择要检查的文档
- 点击「执行检查」
- 查看风险等级和整改建议
- 导出检查报告

### 5.3 制度咨询
- 提供智能制度检索和解读服务

### 5.4 客户经理助手
- 生成营销方案
- 分析客户画像
- 推荐合适产品

## 6. 工作流编排

### 6.1 创建工作流
- 进入「工作流编排」页面
- 修改工作流名称
- 从左侧拖拽节点到画布
- 配置节点属性
- 连接节点
- 点击「保存」

### 6.2 测试工作流
- 点击「测试运行」按钮
- 查看执行结果

## 7. 系统管理

### 7.1 用户管理
- 查看用户列表
- 新增/编辑用户
- 重置用户密码
- 启用/禁用用户
- 批量删除用户

### 7.2 角色管理
- 查看角色列表
- 新增/编辑角色
- 配置角色权限
- 删除角色

### 7.3 租户管理
- 查看租户列表
- 新增/编辑租户
- 配置租户配额
- 删除租户

### 7.4 审计日志
- 查看系统操作日志
- 按用户名、操作类型筛选
- 按日期范围筛选
- 导出日志

### 7.5 系统配置
- 修改系统名称和描述

## 8. 运维监控
