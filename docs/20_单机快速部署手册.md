## 2. 快速部署步骤

### 2.1 克隆项目

```bash
git clone https://github.com/finrag4j/finrag4j.git
cd finrag4j
```

### 2.2 配置环境变量

```bash
cp .env.example .env
# 编辑.env文件，配置数据库密码等
```

### 2.3 一键启动

```bash
cd deploy
docker-compose up -d
```

### 2.4 检查服务状态

```bash
docker-compose ps
```

### 2.5 访问服务

| 服务 | 地址 |
|------|------|
| 前端界面 | http://localhost |
| API文档 | http://localhost:8080/doc.html |
| MinIO控制台 | http://localhost:9001 |

## 3. 初始化配置

### 3.1 创建知识库

```bash
curl -X POST "http://localhost:8080/api/kb?tenantId=1" \
  -H "Content-Type: application/json" \
  -d '{
    "kbName": "金融法规知识库",
    "kbCode": "finance_regulation",
    "description": "金融监管法规知识库"
  }'
```

### 3.2 上传测试文档

通过前端界面上传测试文档，或使用API：

```bash
curl -X POST "http://localhost:8080/api/documents/upload?tenantId=1" \
  -F "file=@test.pdf"
```

### 3.3 测试问答

```bash
curl -X POST "http://localhost:8080/api/chat/new" \
  -d "question=测试问题&kbId=1&tenantId=1&userId=1"
```

## 4. 常见问题

### Q1: 服务启动失败

检查端口占用：
```bash
netstat -tlnp | grep -E "80|8080|5432|6379|9000"
```

### Q2: 数据库连接失败

检查PostgreSQL状态：
```bash
docker-compose logs postgres
```

### Q3: 模型加载失败

检查Ollama服务：
```bash
docker-compose logs ollama
```

## 快速启动

### 方法一: 一键启动 (推荐)
```bash
# 在项目根目录执行
cd d:\coding\github\finrag4j
start.bat
```

### 方法二: 分步启动

#### 1. 启动后端

```bash
# 编译后端
cd d:\coding\github\finrag4j\backend
build.bat

# 启动后端服务
run.bat
```

#### 2. 启动前端

```bash
# 安装依赖并启动
cd d:\coding\github\finrag4j\frontend
run.bat
```

#### 3. 启动Python服务

```bash
cd d:\coding\github\finrag4j\backend\finrag4j-python
.venv\Scripts\python.exe main.py
```

## 常见问题

### 1. 端口被占用
```bash
# 查看端口占用
netstat -ano | findstr ":8080"
# 结束进程
taskkill /F /PID <进程ID>
```

### 2. 后端编译失败
```bash
# 清理Maven缓存
cd backend
%MAVEN_HOME%\bin\mvn.cmd clean install -U
```

### 3. 前端依赖安装失败
```bash
# 清理npm缓存
npm cache clean --force
# 重新安装
cd frontend
rmdir /s /q node_modules
npm install
```

### 4. Python服务启动失败
```bash
# 检查虚拟环境
cd backend\finrag4j-python
.venv\Scripts\python.exe -m pip install -r requirements.txt
```

## 开发说明

### 前端开发
```bash
cd frontend
npm run dev    # 开发模式，热重载
npm run build # 生产构建
```

### 后端开发
使用IDE（如IntelliJ IDEA）打开backend目录，配置:
- JDK: 21
- Maven: 3.9.9
- 工作目录: backend

### Python开发
```bash
cd backend\finrag4j-python
.venv\Scripts\python.exe main.py  # 热重载模式
```
