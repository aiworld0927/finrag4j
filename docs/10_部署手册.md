# 10_部署手册

## 1. 环境要求

- Docker 24+
- Docker Compose 2+
- Kubernetes 1.28+（可选）

## 2. 单机部署

### 2.1 使用Docker Compose

```bash
cd deploy/docker-compose
docker-compose up -d
```

### 2.2 环境变量配置

| 变量 | 说明 | 默认值 |
|------|------|--------|
| POSTGRES_HOST | 数据库地址 | localhost |
| POSTGRES_PORT | 数据库端口 | 5432 |
| POSTGRES_DB | 数据库名称 | finrag4j |
| REDIS_HOST | Redis地址 | localhost |

## 3. 集群部署

### 3.1 使用Kubernetes

```bash
cd deploy/k8s
kubectl apply -f namespace.yaml
kubectl apply -f postgres/
kubectl apply -f redis/
kubectl apply -f finrag4j/
```

### 3.2 配置说明

- 副本数：3个
- 资源限制：CPU 2核，内存4GB
- 存储：PersistentVolume

## 4. 服务启动验证

```bash
# 检查服务状态
curl http://localhost:8080/api/health

# 检查Python服务
curl http://localhost:8001/health
```
