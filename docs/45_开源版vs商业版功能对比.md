# FinRag4j 开源版 vs 商业版 功能对比

## 文档概述

本文档详细对比FinRag4j开源免费版和商业企业版的功能差异，帮助用户选择合适的版本。

**文档版本**: v1.0.0

---

## 1. 版本概述

### 1.1 开源版

- **定位**: 面向个人开发者、中小企业的基础RAG应用
- **授权**: Apache 2.0 开源协议
- **费用**: 完全免费
- **支持**: 社区支持

### 1.2 商业版

- **定位**: 面向金融机构、大型企业的企业级应用
- **授权**: 商业授权
- **费用**: 按需定价
- **支持**: 企业级技术支持

## 3. 适用场景

### 3.1 开源版适用场景

- 个人学习研究
- 中小企业内部知识库
- POC概念验证
- 开发测试环境
- 非关键业务场景

### 3.2 商业版适用场景

- 金融机构生产环境
- 大型企业知识管理
- 多部门/多子公司场景
- 强合规审计要求
- 关键业务系统

---

## 4. 升级路径

### 4.1 从开源版升级到商业版

1. 联系商务获取商业版授权
2. 执行数据库升级脚本
3. 替换商业版JAR包
4. 配置商业版功能参数
5. 重启服务

### 4.2 数据迁移

开源版数据可无缝迁移到商业版：
- 数据库结构兼容
- 向量数据保留
- 知识库配置保留
