# FinRag4j 快速启动指南

## 环境要求

### 必需组件
- **Java JDK**: `C:\dev\Java\jdk-21.0.11`
- **Maven**: `C:\dev\apache-maven-3.9.9`
- **Node.js**: v16+
- **Python**: 3.9+

### 中间件服务
- **Nacos**: localhost:8848 (服务发现与配置中心)
- **PostgreSQL**: localhost:5432 (数据库)
- **Redis**: localhost:6379 (缓存)
- **MinIO**: localhost:9000 (对象存储)

## 快速启动

### 方法一: 一键启动 (推荐)
```bash
# 在项目根目录执行
cd d:\coding\github\finrag4j
start.bat
```

### 方法二: 分步启动

#### 1. 启动后端

```bash
# 编译后端
cd d:\coding\github\finrag4j\backend
build.bat

# 启动后端服务
run.bat
```

#### 2. 启动前端

```bash
# 安装依赖并启动
cd d:\coding\github\finrag4j\frontend
run.bat
```

#### 3. 启动Python服务

```bash
cd d:\coding\github\finrag4j\backend\finrag4j-python
.venv\Scripts\python.exe main.py
```

## 常见问题

### 1. 端口被占用
```bash
# 查看端口占用
netstat -ano | findstr ":8080"
# 结束进程
taskkill /F /PID <进程ID>
```

### 2. 后端编译失败
```bash
# 清理Maven缓存
cd backend
%MAVEN_HOME%\bin\mvn.cmd clean install -U
```

### 3. 前端依赖安装失败
```bash
# 清理npm缓存
npm cache clean --force
# 重新安装
cd frontend
rmdir /s /q node_modules
npm install
```

### 4. Python服务启动失败
```bash
# 检查虚拟环境
cd backend\finrag4j-python
.venv\Scripts\python.exe -m pip install -r requirements.txt
```

## 开发说明

### 前端开发
```bash
cd frontend
npm run dev    # 开发模式，热重载
npm run build # 生产构建
```

### 后端开发
使用IDE（如IntelliJ IDEA）打开backend目录，配置:
- JDK: 21
- Maven: 3.9.9
- 工作目录: backend

### Python开发
```bash
cd backend\finrag4j-python
.venv\Scripts\python.exe main.py  # 热重载模式
```
