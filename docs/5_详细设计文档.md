# 5_详细设计文档

## 1. 文档解析模块

### 1.1 DocumentParser类
- 方法：parsePdf(file) → str
- 方法：parseWord(file) → str
- 方法：parseExcel(file) → str
- 方法：parseTxt(file) → str

## 2. OCR服务模块

### 2.1 OCRService类
- 方法：recognize(image) → str
- 方法：batchRecognize(images) → List[str]

## 3. 文本分块模块

### 3.1 TextChunker类
- 方法：chunk(text, strategy, size, overlap) → List[Chunk]

### 3.2 分块策略
- REGULATORY: 600字符/块，重叠100字符
- CONTRACT: 800字符/块，重叠150字符
- NOTICE: 400字符/块，重叠50字符

## 4. API接口设计

### 4.1 文档解析接口
- POST /api/parse
- 参数：file, fileType
- 返回：ParseResponse

### 4.2 OCR识别接口
- POST /api/ocr
- 参数：image
- 返回：OCRResponse

### 4.3 文本分块接口
- POST /api/chunk
- 参数：text, strategy, chunkSize, chunkOverlap
- 返回：ChunkResponse
