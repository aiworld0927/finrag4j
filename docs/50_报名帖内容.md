# FinRag4j 项目报名帖

## 项目概述

FinRag4j 是一个面向金融行业的企业级大模型RAG应用框架，旨在提供开箱即用的金融垂直领域知识库问答解决方案。

---

## 项目亮点

### 🎯 核心价值
- **金融垂直**：预置金融知识库、合规检查、信贷抽取等金融场景能力
- **私有化部署**：数据不出域，满足金融行业合规要求
- **开箱即用**：完整前后端、一键部署、快速落地
- **灵活扩展**：工作流编排、Agent定制、模型路由

### 🛠️ 技术栈
- **后端**：Java 21 + SpringBoot 3.2 + Spring Cloud
- **前端**：Vue 3 + Vite + Element Plus
- **数据库**：PostgreSQL 15 + PGVector
- **缓存**：Redis 7
- **消息队列**：RocketMQ
- **对象存储**：MinIO
- **Python服务**：FastAPI + PyTorch

### 📊 功能模块
1. **认证授权模块**：JWT认证、RBAC权限管理
2. **知识库管理模块**：文档上传、解析、分块、向量化
3. **RAG问答模块**：智能问答、多轮对话、来源溯源
4. **金融Agent模块**：合规自查、合同抽取、制度咨询
5. **系统管理模块**：用户管理、角色管理、配置管理

## 参与方式

### 🚀 快速开始
```bash
# 克隆项目
git clone https://github.com/finrag4j/finrag4j.git

# 进入项目目录
cd finrag4j

# 一键启动（需要Docker）
./scripts/start.sh
```
