### 3.2 核心流程

#### 文档上传流程

```
1. 用户上传文件 → DocumentController.upload()
2. 验证文件类型和大小
3. 保存文件到MinIO
4. 创建Document记录到数据库
5. 发送解析任务到RocketMQ
6. 返回上传结果

7. Python服务消费消息
8. 解析文档 → 文本提取 → 分块 → 向量化
9. 保存向量到PGVector
10. 更新Document状态
```

### 4.2 检索流程

```
1. 用户提交查询 → SearchController.search()
2. 文本向量化 → TextEmbeddingService.embed()
3. 向量相似度检索 → VectorSearchService.search()
4. 结果排序和过滤 → SearchResultService.process()
5. 返回检索结果
```

### 4.3 向量检索SQL示例

```sql
-- 余弦相似度检索
SELECT 
    id, 
    content, 
    document_id, 
    embedding <-> $1::vector AS distance
FROM vector_chunk
WHERE knowledge_base_id = $2
ORDER BY embedding <-> $1::vector
LIMIT $3;

-- IVFFlat索引优化
CREATE INDEX idx_vector_chunk_embedding 
ON vector_chunk 
USING ivfflat (embedding vector_cosine_ops) 
WITH (lists = 100);
```
