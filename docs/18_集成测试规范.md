## 1. 集成测试范围

### 1.1 测试分层

```
┌─────────────────────────────────────────────┐
│            E2E 端到端测试                    │
│    完整业务流程（上传->解析->检索->问答）     │
├─────────────────────────────────────────────┤
│            服务集成测试                      │
│    模块间接口（Auth->Gateway->Document）     │
├─────────────────────────────────────────────┤
│            组件集成测试                      │
│    数据库/缓存/消息队列交互                   │
├─────────────────────────────────────────────┤
│            单元测试                          │
│    单个类/方法逻辑                           │

```

### 1.2 必测场景

| 测试类型 | 覆盖范围 | 优先级 |
|---------|---------|--------|
| 服务间调用 | Gateway 路由转发到各微服务 | P0 |
| 数据库操作 | CRUD + 事务 + 并发 | P0 |
| 缓存集成 | Redis 读写 + 过期策略 | P1 |

## 3. 测试用例设计

### 3.1 接口集成测试

```java
@SpringBootTest(webEnvironment = SpringBootTest.WebEnvironment.RANDOM_PORT)
@AutoConfigureWebTestClient
class GatewayIntegrationTest {
    
    @Autowired
    private WebTestClient webClient;
    
    @Test
    @DisplayName("完整鉴权链路：注册->登录->访问受保护接口")
    void testFullAuthFlow() {
        // Step 1: 注册用户
        webClient.post().uri("/api/auth/register")
            .bodyValue(registerRequest)
            .exchange()
            .expectStatus().isOk();
        
        // Step 2: 登录获取Token
        String token = webClient.post().uri("/api/auth/login")
            .bodyValue(loginRequest)
            .exchange()
            .expectStatus().isOk()
            .returnResult(LoginResponse.class)
            .getResponseBody()
            .blockFirst()
            .getData()
            .getAccessToken();
        
        // Step 3: 使用Token访问受保护接口
        webClient.get().uri("/api/documents")
            .header("Authorization", "Bearer " + token)
            .exchange()
            .expectStatus().isOk();
    }
}
```

### 3.2 数据库事务测试

```java
@Test
@DisplayName("文档上传：元数据入库 + 文件存储原子性")
void testDocumentUploadAtomicity() {
    // 模拟 MinIO 上传失败
    when(minioClient.putObject(any()))
        .thenThrow(new RuntimeException("Storage error"));
    
    // 验证事务回滚
    assertThatThrownBy(() -> documentService.upload(file))
        .isInstanceOf(RuntimeException.class);
    
    // 验证数据库无记录
    assertThat(documentMapper.selectCount(null)).isZero();
}
```

### 3.3 消息队列测试

```java
@Test
@DisplayName("文档解析：上传后触发异步解析任务")
void testDocumentParseAsync() {
    // 上传文档
    Document doc = documentService.upload(testFile);
    
    // 验证消息已发送
    await().atMost(Duration.ofSeconds(5))
        .untilAsserted(() -> {
            verify(rocketMQTemplate).asyncSend(
                eq("document-parse-topic"),
                any(Message.class),
                any(SendCallback.class)
            );
        });
    
    // 模拟消费端处理
    parseConsumer.onMessage(parseMessage);
    
    // 验证状态更新
    Document updated = documentMapper.selectById(doc.getId());
    assertThat(updated.getStatus()).isEqualTo("processed");
}
```

## 7. 执行策略

### 7.1 测试阶段

| 阶段 | 触发条件 | 测试范围 | 时长 |
|------|---------|---------|------|
| 本地提交前 | git pre-commit | 单元测试 | <2min |

### 7.2 Maven 配置

```xml

        <id>integration-test</id>
        <build>
            <plugins>
                <plugin>
                    <groupId>org.apache.maven.plugins</groupId>
                    <artifactId>maven-failsafe-plugin</artifactId>

                            <include>**/*E2ETest.java</include>

                    </configuration>

```

## 8. 问题排查

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|---------|
| Testcontainers 启动失败 | Docker 未运行 | 检查 Docker 服务状态 |
| 端口冲突 | 随机端口未生效 | 使用 @DynamicPropertySource |
| 测试数据污染 | 事务未回滚 | 添加 @Transactional |
| 异步测试失败 | 等待时间不足 | 使用 Awaitility 等待 |
| 时区断言失败 | 数据库/JVM 时区不一致 | 统一设置 spring.jpa.properties.hibernate.jdbc.time_zone=UTC |
